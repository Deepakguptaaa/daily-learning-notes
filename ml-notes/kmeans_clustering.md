# K-Means Clustering

## What is K-Means?

K-Means is an unsupervised machine learning algorithm used to group similar data points into clusters.

## How It Works

1. Choose K clusters.
2. Randomly initialize centroids.
3. Assign points to nearest centroid.
4. Update centroids.
5. Repeat until convergence.

## Real-World Example

Customer Segmentation:
- Group customers based on spending habits.
- Used in marketing and recommendation systems.

## Advantages

- Simple and fast
- Easy to understand
- Works well on large datasets

## Limitations

- Need to choose K beforehand
- Sensitive to outliers

## Applications

- Customer segmentation
- Image compression
- Market analysis

## What I Learned

- K-Means is an unsupervised algorithm.
- It groups similar data into clusters.
- The value of K determines the number of clusters.
