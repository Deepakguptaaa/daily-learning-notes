# Principal Component Analysis (PCA)

## What is PCA?

Principal Component Analysis (PCA) is a dimensionality reduction technique used to reduce the number of features while preserving important information.

## Why Use PCA?

- Reduces complexity
- Speeds up training
- Helps visualize high-dimensional data
- Reduces noise

## How PCA Works

1. Standardize the data
2. Compute covariance matrix
3. Calculate eigenvectors and eigenvalues
4. Select principal components
5. Transform the data

## Real-World Example

Suppose a dataset has 100 features.

PCA can reduce it to 10–20 important features while retaining most of the information.

## Advantages

- Faster training
- Less storage required
- Reduces overfitting

## Limitations

- Reduced interpretability
- Some information is lost

## Applications

- Image compression
- Face recognition
- Data visualization
- Feature engineering
